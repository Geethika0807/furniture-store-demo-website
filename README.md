# Furniture Store Demo Website

A modern and responsive furniture store website built using HTML and CSS.

## Features

- Responsive Design
- Product Showcase
- Customer Reviews
- Image Gallery
- WhatsApp Contact Button
- Contact Section
- Mobile-Friendly Layout

## Technologies Used

- HTML5
- CSS3

## Purpose

This project was created as a demo website for furniture businesses to showcase products and improve their online presence.
